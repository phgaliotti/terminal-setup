# Setup rápido

Backup e setup rápido das minhas configurações de terminal (Zsh, Powerlevel10k, Oh My Zsh, etc).

## Instalação rápida

```bash
git clone https://github.com/seuusuario/dotfiles.git
cd terminal-setup
./setup.sh
```

## O que está incluso?
- .zshrc customizado
- Powerlevel10k
- Aliases e funções pessoais
- Plugin Oh My Zsh

## Recomendações
- Instale o Zsh e o Git antes de rodar o setup
